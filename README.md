#  Travel & Tourism Management System (TravelEase)

A Java-based desktop application developed to simplify and automate travel and tourism management processes.  
This system centralizes customer management, hotel booking, tour packages, and payment handling into a single integrated platform.


##  Project Overview

The **Travel & Tourism Management System** is designed to manage:

- Customer Registration
- Tour Package Booking
- Hotel Reservation
- Payment Processing
- Booking Cancellation
- Administrative Data Management

The system reduces manual paperwork, improves booking efficiency, and enhances user experience through a user-friendly interface.

## Objectives

- Automate travel agency operations
- Provide centralized booking management
- Reduce manual documentation
- Improve customer satisfaction
- Maintain secure and structured database reco
